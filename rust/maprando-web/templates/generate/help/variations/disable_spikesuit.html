Disable spikesuits

When this option is enabled, the game will be modified so that Samus is no longer able to obtain a flash suit via a glitched shinespark (by a spikesuit, slopespark, or R-mode knockback spark). Any logic that would require the use of these techniques will also be disabled.

A flash suit is a glitched state in which Samus is in a permanent state of shinecharge. A flash suit may still be obtained by other methods, by interrupting a Crystal Flash using an elevator or R-mode stand-up.
