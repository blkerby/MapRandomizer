Remove spikesuit ability

If enabled, the game will be modified so Samus cannot gain a flashsuit from a glitched shinespark.

This setting will also disable certain parts of flashsuit logic if they would be in logic based on selected difficulty (Spikesuit / Slopespark / R-Mode knockback Spark). A flash suit may still be gained via Elevator CF or R-Mode CF Interrupt, even if disabled in logic.
