Speed Booster

This affects the Speed Booster item.

  • Vanilla: Speedbooster is a single collectible item and functions identically to the vanilla game.
  • Split: Vanilla Speed Booster is replaced with two new collectible items, "Spark Booster" and "Blue Booster".

This setting is taken into account in the logic.

  • Blue Booster
    • : This item allows Samus to perform Temporary blue chains and gain run speed but not shinespark.

  • Spark Booster
    • : This item allows samus to shinespark, spikesuit (if not disabled) but without the ability to perform any of the blue speed abilities that Blue Booster enables.
