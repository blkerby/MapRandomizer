Enemies cleared during escape

If this option is enabled, enemies do not spawn during the escape.

If this option is disabled, in many rooms enemies will cause heavy lag and visual glitches during the escape (much of which is vanilla game behavior but not normally observable in casual play).

Note that regardless of whether or not this option is enabled, the randomizer opens up barriers during the escape as follows:

  • All bosses/minibosses are cleared.
  • Shaktool Room is cleared.
  • Acid Chozo Statue acid is drained.
  • Maridia Tube is broken.
