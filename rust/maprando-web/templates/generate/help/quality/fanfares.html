Item fanfares

Controls the time it takes for the dialogue box that pops up upon obtaining an item to close.

  • Vanilla: Exactly as in the original game. Dialogue box lasts for 360 frames before it can be closed.
  • Trimmed: The fanfare plays as usual, but the roughly two seconds of silence afterward is removed, so the box can be closed sooner. Box lasts 256 frames.
  • Off: No fanfare is played. Instead, a short sound effect which depends on the item is played. Box lasts 16 frames.
