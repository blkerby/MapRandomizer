Enhanced save icons shown on map

This option affects how the pause menu map appears:

  • Enabled: Save icons are enhanced. The 3 most recently used save stations are color coded: yellow (most recent save), orange (previous save) and pink (earliest save / next to be overwritten). Samus's spawn point is also shown as a helmet until 3 save stations have been used.
  • Disabled: Saves stations are shown as regular map icons and Samus' spawn point as a helmet icon are not drawn on the map.
