Save icons highlighted on map

This option affects how the pause menu map appears:

  • Enabled: Save icons are enhanced. The 3 most recently used save stations are color coded: yellow, oranage and pink. Samus's spawn point is also shown as a helmet until 3 save stations have been used.
  • Disabled: Saves are shown as regular save stations and Samus' spawn point as a helmet icon are not drawn on the map.
