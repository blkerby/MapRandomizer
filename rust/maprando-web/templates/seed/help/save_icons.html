Save icons

This option affects how the pause menu map appears:

  • Enabled: The three most recently used save stations are shown, color coded as follows: yellow for the most recent save (same appearance as in the vanilla game), orange for the previous save, and pink for the next to be overwritten. Samus's spawn point is also shown as a helmet until three saves have been used. An icon is also drawn for the Ship, with the same appearance as in the vanilla game.
  • Disabled: Icons are not drawn for saves, the spawn point, nor the Ship.
