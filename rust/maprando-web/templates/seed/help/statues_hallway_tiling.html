Statues Hallway tiling

This setting controls the graphical appearance of rooms that connect to the Tourian area.

  • Disabled: No special tiling is applied to these rooms. Their tiling will follow the same "Tile theme" setting used in the rest of the game.

  • Default: Statues Hallway tile theme is applied to these rooms only if the "Tile theme" is set to "Area-themed" or "Scrambled".

  • Enabled: Statues Hallway tile theme is always applied to these rooms.
